# Support

OpenNivara is alpha software maintained in public on GitHub.

- Normal bugs: open a public issue with non-sensitive reproduction details.
- Feature requests: open a public feature-request issue.
- Contribution questions: open an issue or comment on an existing contributor
  task.
- Security vulnerabilities: do not open a public issue. Follow
  [SECURITY.md](SECURITY.md).

Never include API keys, bot tokens, private memories, local databases, `.env`
files, personal files, or sensitive logs in public issues.
