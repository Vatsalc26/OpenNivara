---
title: "OpenNivara"
description: "Local-first desktop assistant with audited memory and safe customization."
---

OpenNivara is a desktop-first personal assistant built with Rust, SQLite, React, and Tauri.

Core principles:

- Desktop is the primary product.
- Store themes only change visuals.
- Settings controls assistant behavior.
- Memory is local-first and has no templates.
- Time, location, memory, workspace, and graph context are included only when relevant and audited.
