---
title: "Docs Governance"
description: "How internal and public docs fit together."
---

`docs/` is the internal canonical engineering tree. `docs-site/` is the public-facing Mintlify site.

Mintlify CLI:

- Official global install: `npm i -g mint`
- Official no-global fallback: `npx mint dev`
- Local preview command from this folder: `mint dev`

Do not commit Mintlify secrets or deployment credentials.
