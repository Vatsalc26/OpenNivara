---
title: "Preferences"
description: "How OpenNivara preferences guide assistant responses without being installed by Store themes."
---

Preferences are explicit user-authored guidance for assistant behavior. They live in local TOML configuration and are controlled by Settings or CLI setup commands.

```bash
cargo run -- init-preferences
cargo run -- preferences-path
cargo run -- show-preferences
```

OpenNivara can select relevant preference sections for a request. Preferences are not Store theme content, and visual themes must not modify them.
