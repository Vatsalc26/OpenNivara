---
title: "Memory Graph"
description: "How OpenNivara derives graph context from local memory while keeping SQLite as the source of truth."
---

The Memory Graph is a rebuildable SQLite-derived index over memory-related data. It helps OpenNivara inspect related memories, facets, entities, tasks, and sources while keeping SQLite as the source of truth.

Graph context is not a separate trust boundary. It can contain sensitive relationships derived from local memory. Treat graph databases and exports as private local state.
