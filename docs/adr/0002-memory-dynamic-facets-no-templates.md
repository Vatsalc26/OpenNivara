# ADR 0002: Memory Dynamic Facets, No Templates

Status: accepted.

Memory is content-driven. OpenNivara must not use Student, Freelancer, Health, Travel, occupation, or role templates to define what memory can store.

Memory facets provide flexible domain/type labels. A single user can have education, work, health, family, travel, technical, household, creative, automation, and custom memories at the same time.
