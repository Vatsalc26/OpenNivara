# Store Skill Packs

The Store can preview built-in skill packs before installation. Skill-pack cards
show skill count, category, exam coverage, and safety badges. The details dialog
lists included skills with best-for notes, sample prompts, and official source
labels when a skill requires fresh information.

Installing a skill pack only copies data into the marketplace. Users must go to
Settings > Skills to enable or disable behavior.

## Safety Messaging

Store skill packs are presented as data-only:

- no executable code
- no tool permission changes
- no install-time behavior activation
- no network tools granted by the pack
