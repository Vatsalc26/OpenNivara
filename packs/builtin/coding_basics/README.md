# Coding Basics Pack

Welcome to the **Coding Basics Pack** for OpenNivara! This pack includes configurations tailored for software engineers and students looking to learn modern programming concepts.

## What's Included

- **Preferences**: Clean, functional, and defensive design options, discouraging hardcoded numbers or deep accumulator loop nestings.
- **Contexts**: Beginner Rust borrow checker rules to help reason through compiler messages.
- **Style Guidelines**: A step-by-step technical response layout.
- **Theme**: Coding Cyan dark interface.
- **Command Snippets**:
  - `Explain Rust compiler error`
  - `Review my code`
