# OpenNivara Roadmap

This roadmap is intentionally lightweight. It describes current contributor
direction without promising timelines.

## Now

- Improve contributor onboarding and community health.
- Maintain curated Student Essentials quality.
- Upgrade India Engineering Exams pack with comprehensive evaluation.

## Next

- Continue pack-by-pack skills quality work.
- Improve public documentation and project showcase material.
- Define a future alpha release-readiness checklist.

## Later

- Broader India-relevant skills based on user feedback.
- Additional desktop UX improvements.
- Distribution and release work when separately approved.
